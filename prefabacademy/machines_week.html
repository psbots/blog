Stickers with Vinyl Cutter

Vinyl Cutter can be used to cut out Vinyl Stickers for labelling or increasing the aesthetic value of the projects done in the FabLab. Luciano told us about the functioning of the Vinyl Cutter and cut out a sample design using the fab modules software on Ubuntu. He then asked us to design or download something to practice operating the machine.

I did a google image search for interesting black and white figures and found this great picture of Iron Man. I had to modify the dimensions of the image and convert it to grayscale.

After opening fab modules, click on image(png) as input and Roland vinylcutter (.camm) as output. Then click on make_png_camm button.

Now click on load image and select the required image. Resize it as per the requirements. Select vinyl in the dropdown list and click on make .path followed by make .camm

Next set-up the Vinyl Cutter by loading the vinyl sheet between the rollers as shown. The roll can be set by selecting "Roll" and then Origin can also be set. Click on "send it" on the fab modules to start cutting the vinyl.

Here's the Vinyl cut out and placed on a masking tape. I put the sticker on my hard disk. Looks pretty great! This machine really makes it very convenient to make labels and graphic stickers for our projects.

3D Printing a Pen Stand

Luciano asked us to design an object within 5cm * 5cm * 5 cm dimensions, which cannot be manufactured subtractively. The idea was to design an intricate enough object that would be difficult to manufacture subtractively but simple enough to 3D Print.

I was brainstorming on what could be built, and came up with a very simple design for a cylindrical Pen Holder with a pattern of rounded rectangular holes around it. I used Antimony to do the design.

First I made a cylinder with the outer radius and subtracted a cylinder to get the required thickness for the stand.

Then I created a rounded cube and subtracted it from the cylinder after placing it in the position I wanted. Then I used the Array Polar to replicate the hole around the surface of the cylinder.

Finally, I copy pasted the Array twice, translated it to the middle and upper positions, rotated it a bit to give it an interesting asymmetry and here’s the final result! Here are the .sb and .stl files :

Now comes the actual 3D printing of the model. First I opened the .stl model in Cura and exported the .gcode with the Normal mode settings.

Then I loaded the .gcode file onto the SD card and gave the Print command on the Ultimaker 2 machine.

Here’s the Pen Holder in action at the Lab. Though it was meant as a “Pen” holder, it is much more useful now near the soldering station for holding Cutters, Scale and Tweezers!

Electronics Production using Modella

The Modela is one of the most used machines in the FabLab. Its a tiny desktop CNC that can mill PCBs, create 3D models on wax and even scan in 3D. The machine is actually a is a 2.5D machine which can also be used for milling wax with limited depth (Z axis). It is preferred for making PCBs using FR-1 copper boards as it doesn’t leave any chemical wastes. But do not use it to mill FR-4 boards as it creates a lot of dust from fibre glass and which is toxic when it is inhaled.

The main steps to mill a pcb are desiging the circuit (export as .png format), using fab modules to controll the machine, setting the machine and milling.

Below you can see the design files. First part is the design output for milling the traces and the second is the outline to cut out the board. If you are using the fab modules, the the design should be in .png format. We took the design from http://fab.cba.mit.edu/content/projects/fabisp which is already done using kokopelli.

Setting up the Modella

There are two milling bit that will be used for Electronics production in modella :

  • 1/64 drill bit - For removing the unwanted copper and get the traces
  • 1/32 drill bit - For cutting the final board

For mounting the bit on the head, there is a screw that is to be loosen first. We can use allen key to loosen it. Then place the bit such that the majority of the bit goes inside the head. Then tighten the screw using the allen key

Next the board is placed on top of the sacrificial layer using two strips of strong double sided tape.

Using fab modules

First open fab modules and select .png as input and .rml as output as shown in the screenshot below.

Load the traces.png file by clicking "load .png" button.

Next we have to home the machine using trial and error to the location that we want to mill. This is done by entering suitable x and y co-ordinate values and clicking on "move to xmin,ymin"

Next step is to set the origin point for the Z axis. To do this, bring the milling bit down by pressing the “Down arrow” button on the machine till it almost reaches the board and then loosen the bit using the Allen key. The bit should slowly fall down and touch the surface of the board, after which the mill bit should be tightened by using the Allen key.

Click on the “Make .rml” button and send the machine file to the Modela by clicking on “Send It!”. After which the machine will start milling and removing copper from around the traces.

Finally load the outline.png, change the milling bit to 1/32 bit following the above instructions and cut out the outline to get the final board

Now its time to solder! First all the required components are listed out on a paper and the components are sourced from the consumables inventory. This allows a hassle free way to solder the components without the need to run around looking for components in between soldering.

I programmed the FabISP following the instructions in this tutorial.

The procedure failed to burn the program initially and asked to check all the connections. I ran through the soldering connections and found a couple of disconnected juntions. I reapplied solder and tried. The FabISP is successfully programmed and ready to program another!

Downloadable Files can be found at http://fab.cba.mit.edu/content/projects/fabisp
